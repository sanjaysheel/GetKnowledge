Updating a Lists


As you know that tuple are immutable, which means you cannot update or change the values of tuple. But you can create new tuples to take portions of the existing tuples. 

 
        >>> tuple1 = (1, 2, 3, 4, 5, 6)
        >>> tuple2 = ('a', 'b', 'c', 'd')
        >>> tuple3 = tuple1 + tuple2
        >>> tuple3
        (1, 2, 3, 4, 5, 6, 'a', 'b', 'c', 'd')
        >>>


Deleting a Lists


In the list you have already learn about del keyword we can use this keyword not only in list but also for tuple. To Remove individual tuple's elements is not possible. But you can remove or delete the tuple by using del keyword.

 
        >>> tuple1 = (1, 2, 3, 4, 5, 6)
        >>> tuple2 = ('a', 'b', 'c', 'd')
        >>> del tuple1
        >>> del tuple2

Uses of max function

Description


It Returns largest value of the list. In the parameter it used to take the tuple type of object and cheak the max value present in the tuple. In simple word this function returns the max number of elements present in the tuple.


Syntex-

    
        max(tuple)

Example-

        >>> tuple2 = ('a', 'b', 'c', 'd')
        >>> max(tuple2)
        'd'
        >>> tuple1 = (1, 2, 3, 4, 5, 6)
        >>> max(tuple1)

        >>>
