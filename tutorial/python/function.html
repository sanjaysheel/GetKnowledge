Scope of variable

Some variables in a program may not be accessible at all locations in that program and some variable in a programm may not be accessible at all locations in that program. This depends on where you have declared a variable. The scope of a variable determines the portion of the program where you can access a particular identifier. There are two type of variable we can define in the python programm.

  1. Global variables
  2. Local variables

Global and local variable

The Variables which are defined inside a function body have a local scope, and those which are defined outside have a global scope. This means that local variables can be accessed only inside the function in wherer they have declared, whereas global variables can be accessed out of the program body by all functions. When you call a function, the variables declared inside it are brought into scope. For example :-

input example :-

            var = 0
            def mul( arg1, arg2 ):
                # Add both the parameters and return them."
                var = arg1 * arg2 # Here total is local variable.
                print ("Inside the function local var : ", var)
                return var

            mul( 10, 20 )
            print ("Outside the function global var : ", var )

Types of argument in Function


you can call a function using the following type

  1. Default arguments
  2. Keyword Arguments
  3. positional arguments
  4. Variable length arguments


Default arguments


A default argument is an argument that assumes a default value if a value is not provided in the function call for that argument. The following example gives an idea on default arguments, it prints default age if it is not passed.

            >>> def fun(name,age,city='indore'):
            ...     '''function with default arguments'''
            ...     return 'My is: ',name,', age is: ',age,' and live in ',city
            ... 
            >>> ret = fun('sanjay' , 23 )
            >>> ret
            ('My is: ', 'sanjay', ', age is: ', 23, ' and live in ', 'indore')
            >>> ret = fun('raju' , 34)
            >>> ret
            ('My is: ', 'raju', ', age is: ', 34, ' and live in ', 'indore')
            >>>

Keyword arguments


In function we you use Keyword arguments in a function call, then the caller identifies check the arguments by the parameter name. It also allows you to place them to respectively them keyword because the Python interpreter is able to use the keywords provided to match the values with parameters. You can also make keyword to place the value to the parameter respertivly. For example- 

              >>> def fun(name,age=23,city='indore'):
            ...     '''function with default arguments'''
            ...     return 'My is: ',name,', age is: ',age,' and live in ',city
            ... 
            >>> ret = fun(city = 'indore' , name = 'jhon')
            >>> ret
            ('My is: ', 'jhon', ', age is: ', 23, ' and live in ', 'indore')
            >>>

The Anonymous Functions


The function is called anonamous function. Which are not declared in the standard manner means by using the def keyword. To create a anonamous fuction You can use the lambda.

  • An anonymous function cannot be a direct call.
  • Lambda functions have their own local namespace and it can not access variables other than those in their parameter list and those in the global namespace.
  • They cannot contain commands or multiple expressions.
  • Lambda function can take any number of arguments but it return only one value in the form of an expression.

Syntex

            lambda [arg1 [,arg2,.....argn]]:expression

example input :-

            inp=int(input("Enter only one number to get square: "))
            square = lambda arg1 : arg1 * arg1
            print("square of ",inp,' is: ' ,square(inp))
