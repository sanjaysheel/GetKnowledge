About List


list is the basic data structure in Python. in which the sequence of element is assigned in its position or index. The first index zero, the second index is one, and so on. Python has six built-in types of sequences, in which most common are lists and tuples. here we can do some operations include indexing, slicing, adding, multiplying, and to check for membership. In addition, Python has built-in functions for finding the length of a sequence, smallest elements in the list and for finding largest element in the list. some examples:

Python list is the most versatile datatype, which can be written as a list of comma-separated values (items) between square brackets. The best things about the is that Lists may contain items of different types.



Creating a Lists


Creating a simple list by putting comma-separated values between square brackets. For example- 

 
        >>> list1 = [1,2,4,5,6]
        >>> list1 
        [1, 2, 4, 5, 6]
        >>>
    
 
        >>> list2 = [1, "indore", "Student", "class"]
        >>> list2
        [1, 'indore', 'Student', 'class']
        >>>


Updating a Lists


You can update the elements of lists by useing the slice on the left-hand side of the assignment operator, and you can add a new elements in a list with the append() method. For example- 

 
        >>> list2 = [1, "indore", "Student", "class"]
        >>> list2
        [1, 'indore', 'Student', 'class']
        >>> list2[2] = 'abc'
        >>> list2
        [1, 'indore', 'abc', 'class']
        >>> print ("list2[1:5]: ", list2[1:5])
        list2[1:5]:  ['indore', 'abc', 'class']
        >>> print(list2[1])
        indore
        >>> print ("New value available at index 3 : ", list2[3])
        New value available at index 3 :  class
        >>> list2.append(12)
        >>> list2
        [1, 'indore', 'abc', 'class', 12]
        >>>


Deleting a Lists


For Deleting or removing a list element, you can use either del statement if you know the that which you want to delte other wise You can also use the remove() method if you do not know exactly which items to delete. For example-

Predefine method in python


Python also provide some predefine method for some operation on the list. We are giving this table to know in short the all types of operation. Which are used to perform in list. At the end of the table we will give the brief intro and clear example of all methods.

S. no. name Description
1 list.index(obj) Returns the lowest index in list that obj appears.
2 list.insert(index, obj) Inserts object obj into list at offset index
3 list.append(obj) Appends object obj to list
4 list.count(obj) Returns count of how many times obj occurs in list
5 list.extend(seq) Appends the contents of seq to list
6 list.pop(obj=list[-1]) Removes and returns last object or obj from list
7 list.remove(obj) Removes object obj from list
8 list.sort([func]) Sorts objects of list, use compare func if given
8 list.reverse() Reverses objects of list in place
