About Time Module


This is veru populer module in puthon. This module provides various functions. Although this module is always available in python, but not the all functions are available on all platforms. Most of the function are defined in this module are called platform C library.
To use this module first you have to import this module by using "import" keyword.

Input
    
        import time;
        a=time.time()
        print("time: ",a)
        print(time.localtime())
