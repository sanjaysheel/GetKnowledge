About operator


Operators are those, which can manipulate the value. Consider the expression 4 + 4 = 8. Here, 4 and 4 are called operands and '+' is called the operator.

Python Arithmetic Operators


S. No. Sign Name Desciption
1 + Addtion Adds values
2 - Subtraction Subtracts values
3 / Division Multiplies values
4 % Modulus Divides left hand operand by right hand operand and returns remainder
5 ** Exponent Performs exponential (power) calculation on operators
6 // Floor Division The division of operands where the result is the quotient in which the digits after the decimal point are removed

Python Bitwise operators


Bitwise operator can works on bits and perform operation on bit's. Assume if 'a = 60' and 'b = 13' Now in binary format this will be as follows:-

    
       |---------------------|
       |    a = 0011 1100    |
       |    b = 0000 1101    |
       |---------------------|
       |---------------------|
       |    a^b = 0011 0001  |
       |    a&b = 0000 1100  |
       |    a|b = 0011 1101  |
       |    ~a = 1100 0011   |
       |---------------------|

S. No. Sign Name Desciption
1 & Binary AND This operator copie a bit to the result, if it exists in both operands.
2 | Binary OR It copie a bit, if it exists in either operand.
3 not (Logical NOT) (Logical NOT) Used to reverse the logical state of its operand.
4 ^ Binary XOR It copie the bit, if it is set in one operand but not both.
5 ~ Binary Complement It is unary it is used to 'flipping' bits.
6 << Binary Left Shift This is a Binary left shift in which left operand’s value is moved left by the number of bits specified by the right operand.
7 >> Binary Right Shift This is a Binary Right shift in which left operand’s value is moved right by the number of bits specified by the right operand. 
        >>> a = 30
        >>> b = 13
        >>> a & b
        12
        >>> a | b
        31
        >>> a ^ b
        19
        >>> ~ a
        -31
        >>> ~b
        -14
        >>> a<>> a>>b
        0
