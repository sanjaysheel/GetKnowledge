Assigning Value to the Variable


Python variable does not need explicitly declaration to reserve the memory. The declaration of the variable happens automatically when you assign a value to a variable. The sign (=) is used to assign values to variables. 

        Python 3.7.1 (default, Dec 14 2018, 19:28:38) 
        [GCC 7.3.0] :: Anaconda, Inc. on linux
        Type "help", "copyright", "credits" or "license" for more information.
        >>> number = 1
        >>> amount = 200.12
        >>> name = "sanjay"
        >>> print (number)
        >>> print (amount)
        >>> print (name)

Here the values 1, 200.12 and "sanjay" are assigned to number, amount, and name variables, respectively. Then it produces the follwing outputâˆ’
