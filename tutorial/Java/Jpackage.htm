Package in java


A java package is a group of similar types of classes, interfaces and sub-packages. Package in java can be categorized in two form, built-in package and user-defined package. There are many built-in packages such as java, lang, awt, javax, swing, net, io, util, sql etc.


Advantage of Java Package


  1. Java package is used to categorize the classes and interfaces so that they can be easily maintained.
  2. Java package provides access protection.
  3. Java package removes naming collision.

Types of Package


There are two types of packages.

  • User define package:The package we create is called user-defined package.

    • #Syntax 

    package nameOfPackage;

  • Built-in package:The already defined package like java.io.*, java.lang.* etc are known as built-in packages.

    • #Syntax 

    import java.util.Scanner.*;

    #Example of Package


    package mypackage;  
public class PackgExample
{  
 public static void main(String args[])
 {  
    System.out.println("This is a package");  
 }  
}  

//output
	This is a package
