Java Constructor


  • A Constructor is a special method which initializes an object immediately upon creation.
  • It has the same name as class name in which it resides and it is syntactically similar to any method.
  • When a constructor is not defined,java executes a default constructor which initializes all numeric members to zero and other types to null or spaces.
  • Once defined, constructor is automatically called immediately after the object is created before new operator completes.
  • Constructor do not hava any return type.

There are two types of Constructor that is-

  • Default Constructor
  • Parametrized Constructor

Default Constructor


  • A default constructor is a constructor that either has no parameters
  • If no user-defined constructor exists for a class A and one is needed, the compiler implicitly declares a default parameterless constructor A::A() .

#Example


public class Test
{
Test()//default constructor
{
System.out.println("This is a default constructor");
}
}
class Demo
{
public static void main(String args[])
{
Test ob=new Test();
}
}

//output-This is a default constructor


Parametrized Constructor


  • A Constructor with arguments(or you can say parameters) is known as Parameterized constructor.
  • We can have any number of Parameterized Constructor in our class.

#Example


public class Student
{
Student(int x)//parametrized constructor
{
System.out.println("This is a parametrized constructor");
}
}
class Demo
{
public static void main(String args[])
{
Student ob = new Student(10);
}
}

//output- This is a parametrized constructor


#Example of Constructor


public class ConstExample
{
ConstExample()
{
System.out.println("default constructor");
}
ConstExample(int x)
{
System.out.println("parametrized constructor");
}
}
class Demo
{
public static void main(String args[])
{
ConstExample ob=new ConstExample();
ConstExample ob=new ConstExample(5);
}
}

//output is-
	default constructor
	parametrized constructor
