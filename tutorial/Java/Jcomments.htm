Java Comments


Comments can be used to explain Java code, and to make it more readable. It can also be used to prevent execution when testing alternative code.

There are two type of comments in java-

  1. Single-ilne comments
  2. Multi-line comments

Single-ilne comments


  • Single-line comments start with two forward slashes (//).
  • Any text between // and the end of the line is ignored by Java (will not be executed).
  • Example in single-line commentes

#Example


public class SingleComments
{
public static void main(string args[])
{
System.out.println("This is single-line comments");//This is Comments
}
}


Multi-line comments


  • Multi-line comments start with /* and ends with */.
  • Any text between /* and */ will be ignored by Java.
  • Example in multi-line commnets

#Example


public class MultiCommnets
{
public static void main(String args[])
{
/*int a=10;
int b=20;
int c=a+b;*/This is Comments
System.out.println("This is multi-line comments");
}
}
