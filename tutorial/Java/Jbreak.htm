Java Break

When a break statement is encounterd inside a loop, the loop is immediately terminated and the program control resumes at the next statement following loop.

  • The java break keyword is used to break loop or switch statement.
  • we can use java break statement in all types of loops such as for loop,while loop and do-while loop.

#Syntax

break;

#Example of Break


public class BreakExample
{
public static void main(String args[])
{
for(int i=1;i<=10;i++)
{
if(i==5)
{
break;
}
System.out.println(i);
}
}
}

//output is-
	1
	2
	3
	4
