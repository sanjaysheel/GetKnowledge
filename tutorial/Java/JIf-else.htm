Java If Statement

The if statement to specify a block of Java code to be executed if a condition is true.

#Syntax

if(condition)
{
//Statement
//Statement
}


Java else Statement

The else statement to specify a block of code to be executed if the condition is false.

#Syntax

if(condition)
{
//Statement
//Statement
}
else
{
//Statement
//Statement
}


Example of If-else

public class IF_else
{
public static void main(String args[])
{
int a=10;
int b=20;
if(a<=20)
{
System.out.println("a is greater");
}
else
{
System.out.println("a is smaller");
}
}
