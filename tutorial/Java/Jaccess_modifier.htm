Access Modifiers in java


There are two type of modifiers in java that is-


  • access modifiers
  • non-access modifiers

    • There are 4 types of access modifiers.

    • private
    • default
    • protected
    • public

    There are various types of non-access modifiers.

    • static
    • abstract
    • synchronized
    • native
    • volatile
    • transient and etc.

    private access modifiers


    The private access modifier is accessible only within class.


    #Example of private access modifers


    class A
{  
private int x=10;  
private void get()
{
System.out.println("Hello java");
}  
}  
public class Demo
{  
 public static void main(String args[])
 {  
   A ob=new A();  
   System.out.println(ob.x);//Compile Time Error  
   obj.get();//Compile Time Error  
   }  
}

    default access modifers


    The default access modifers are accessible only in the same package


    #Example of default access Modifiers


    Addition.java


    package xyzpackage;

public class Addition 
{
     int addTwoNumbers(int a, int b)
	 {
	return a+b;
   }
}

    AddExample.java


    package abcpackage;

import xyzpackage.*;
public class AddExample
{
   public static void main(String args[])
   {
	Addition obj = new Addition();
     obj.addTwoNumbers(112, 212);
   }
}

    protected access modifers


    Protected data member and method are only accessible by the classes of the same package and the subclasses present in any package.You can also say that the protected access modifier is similar to default access modifier with one exception that it has visibility in sub classes.

    Classes cannot be declared protected. This access modifier is generally used in a parent child relationship.


    #Example of protected access modifers


    Multiply.java


    package abcpackage;
public class Multiply
 {
   protected int mulTwoNumbers(int a, int b)
   {
	return a*b;
   }
}

    Demo.java


    package xyzpackage;
import abcpackage.*;
class Demo extends Multiply
{
   public static void main(String args[])
   {
	Test ob = new Test();
	System.out.println(ob.mulTwoNumbers(10, 22));
   }
}

//output
	33
